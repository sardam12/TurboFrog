🐸 TurboFrog ($TFG)
About the Project

TurboFrog is a lightweight cryptocurrency client developed in Go, designed for speed, low memory usage, and efficient network performance.
📥 How to Download and Run (Windows)

    Download: Go to the Releases section on the right side of this repository and download TurboFrog.exe.

    Run: Once downloaded, locate TurboFrog.exe in your Downloads folder and double-click it to launch the application.
